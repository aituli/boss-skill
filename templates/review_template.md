# 方案汇报预审模板

## 使用场景
准备向老板汇报方案前，预判老板可能的反应。

## 输入
- 方案名称/主题
- 方案内容简述
- 预期目标

## 输出结构

### 1. 可能的质疑
基于老板的决策风格，列出最可能被问到的问题：

**数据驱动型老板：**
- 「数据呢？这个结论的数据支撑是什么？」
- 「有没有对照数据？benchmark是什么？」
- 「这个数据的置信度是多少？」

**直觉型老板：**
- 「你的直觉是什么？」
- 「这个方案感觉对吗？」
- 「如果是你，你会怎么选？」

**谨慎型老板：**
- 「最坏的情况是什么？」
- 「Plan B是什么？」
- 「如果失败了，怎么止损？」

### 2. 需要准备的材料
- [ ] 详细的数据支撑
- [ ] 至少3个方案对比
- [ ] 风险评估和应对措施
- [ ] ROI测算
- [ ] 执行时间表

### 3. 建议强调的重点
- 量化成果
- 风险控制
- 与战略的契合度

### 4. 可能的踩雷点
- 用"我觉得"开头
- 没有数据支撑
- 只说好的不说风险
- 没有备选方案

### 5. 通过概率评估
基于方案质量、老板风格、当前时机综合评估。
